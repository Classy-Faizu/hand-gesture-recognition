# Hand Gesture Recognition System

## Overview

This project demonstrates a hand gesture recognition system developed using Google Teachable Machine. A custom image dataset was used to train a machine learning model capable of recognising different hand gestures in real time.

## Technologies Used

- Google Teachable Machine
- Machine Learning
- Image Classification

## Features

- Real-time hand gesture recognition
- Custom-trained image classification model
- Image-based gesture prediction
